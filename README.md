# 项目简介

## 核心功能
- 基于YOLOv8的煤矸石检测系统
- 支持图片、视频及摄像头进行检测
- 支持选择不同模型
- 界面实时显示目标位置、目标总数、置信度等信息
- 支持批量检测并直接查看所有检测结果
- 支持检测结果保存
- 可自定义置信度和IOU阈值
- 可选择是否显示标签和原图
- 提供2800张标注数据集，包含煤和煤矸石两类

## 适用场景
- 煤炭行业中的煤矸石自动分拣
- 矿山生产过程中的质量检测
- 煤炭加工厂的原料筛选
- 煤炭质量监控与研究

## 技术特点
- 使用Python3.9开发，依赖opencv-python、PyQt5、ultralytics、torch1.9等技术栈
- 提供完整的源码、数据集和训练模型
- 配有详细的项目运行教程
- 支持Pycharm+Anaconda或VSCode+Anaconda开发环境
- 包含UI界面源文件和图标资源
